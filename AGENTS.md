# AGENTS.md

## このリポジトリの目的

このリポジトリは、ともどの個人プロフィールページを管理するためのもの。

このページは、技術コミュニティ・勉強会・LTイベント・カンファレンスなどで人と会ったとき、NFCカードを読み取ってもらって開くことを想定している。相手が「Xやっていますか？」などと聞いたときに、カードを読み取ってもらえば、このページからX、Zenn、Speaker Deck、GitHubなどへすぐ移動できる。

## 想定ユースケース

- 技術コミュニティで初対面の人に自己紹介する
- SNSや技術発信のアカウントをその場で共有する
- NFCカードを読み取った相手が、迷わず目的のリンクへ移動する

## 設計方針

- モバイルで素早く開けることを優先する
- 最初の画面で「誰のページか」と「どこへ飛べるか」が分かるようにする
- 長い経歴や重いナビゲーションは増やしすぎない
- 文言やUIの変更は、NFCカード経由で初対面の人が見る場面を前提に判断する
- 主要導線は外部プロフィールへのリンク、特にXへの導線

## 実装メモ

- Next.js 製のプロフィールページ
- 主要な表示は `components/profile.tsx`
- メタデータは `app/layout.tsx`
- UIや見た目に関わる変更をしたら、必ずChrome DevToolsで実際の表示を確認する

## コマンド

```bash
pnpm dev
pnpm build
pnpm lint
```
